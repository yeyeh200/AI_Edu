# 用户故事：教师查看教学质量评估报告

**用户故事标题：** 教师查看个人教学质量评估报告
**优先级：** 高
**关联功能需求：** FR-001
**预估工作量：** 5人天
**创建日期：** 2025-11-23
**最后更新：** 2025-11-23

---

## 1. 基本信息

### 1.1 用户故事描述
**作为一个** 教师
**我希望** 查看个人教学质量综合评估报告
**以便于** 了解自己的教学表现，获得数据支撑的教学改进指导

### 1.2 用户角色定义
- **主要用户角色：** 教师（TEA）- 承担各类课程教学任务的教师人员
- **次要用户角色：** 督导专家（SUP）- 查看所负责教师评价结果
- **使用场景：** 教师定期查看教学质量评估结果，了解教学表现，获取改进建议

### 1.3 业务价值
- **用户价值：** 为教师提供客观、全面的教学质量评估，帮助发现优势和不足
- **业务价值：** 提升教师教学意识，促进教学质量持续改进
- **衡量指标：** 教师查看率≥90%，改进建议采纳率≥70%

---

## 2. 验收标准 (Acceptance Criteria)

### 2.1 主要验收标准
**Given** 教师已登录系统且具有查看权限
**When** 教师点击"教学质量评估"菜单项
**Then** 系统应该显示该教师的个人教学质量评估报告
**And** 报告包含综合评分、各维度得分、排名信息、历史趋势等关键信息

### 2.2 详细验收条件
- [ ] **登录验证：** 教师使用学校统一身份认证系统成功登录
- [ ] **权限检查：** 系统验证教师只能查看自己的评价结果
- [ ] **报告显示：** 显示完整的教学质量评估报告，包含所有必要信息
- [ ] **数据准确性：** 报告中的数据与实际计算结果完全一致
- [ ] **响应时间：** 报告生成和显示时间不超过3秒
- [ ] **界面友好：** 报告界面美观专业，信息层次清晰
- [ ] **交互操作：** 支持报告的缩放、打印、导出等操作

### 2.3 边界条件测试
- **正常流程：** 教师有完整教学数据时的正常查看流程
- **异常处理：** 教师教学数据不足时的提示信息
- **边界值测试：** 新教师首次查看、数据为空的情况处理

---

## 3. 业务规则

### 3.1 功能规则
- **评价周期：** 支持按周、月、学期三个时间粒度查看评估
- **数据阈值：** 当活跃度数据少于5次时，不进行评估并给出提示
- **权重配置：** 不同课程类型可调整评估维度权重
- **结果展示：** 教师只能查看自己的评价结果，督导专家可查看负责范围
- **数据脱敏：** 涉及其他教师的对比数据进行适当脱敏处理

### 3.2 数据规则
- **数据完整性：** 需要至少一周的教学活动数据才能生成评估
- **数据准确性：** 评估结果需经过多重验证确保准确性
- **数据时效性：** 评估结果24小时内更新一次
- **数据一致性：** 与其他系统的评价结果保持一致

### 3.3 操作约束
- **权限控制：** 教师只能查看个人评价，不能查看其他教师数据
- **操作频率：** 评估报告每天最多生成3次，防止系统滥用
- **访问时间：** 工作日8:00-18:00可查看详细报告，其他时间查看简化版

## 4. 交互流程

### 4.1 用户操作流程
1. **步骤1：** 教师登录系统，进入个人工作台
2. **步骤2：** 点击导航菜单中的"教学质量评估"
3. **步骤3：** 选择查看的时间周期（周/月/学期）
4. **步骤4：** 系统加载并显示评估报告
5. **步骤5：** 教师查看报告内容，可进行交互操作

### 4.2 系统响应流程
1. **响应1：** 验证用户身份和权限
2. **响应2：** 获取教师的最新教学质量评估数据
3. **响应3：** 生成可视化的评估报告界面
4. **响应4：** 提供报告的打印、导出、分享功能

### 4.3 异常处理流程
- **异常情况1：** 教师数据不足时显示"暂无评估数据"并提供数据补充指导
- **异常情况2：** 系统繁忙时显示加载动画，超过30秒提供重试选项
- **异常情况3：** 权限异常时显示权限不足提示并联系管理员

## 5. 数据需求

### 5.1 输入数据要求
| 数据项 | 数据类型 | 是否必填 | 格式要求 | 取值范围 | 说明 |
|--------|----------|----------|----------|----------|------|
| 教师工号 | String | 是 | 工号格式 | 学校标准工号 | 唯一标识教师身份 |
| 查询周期 | String | 是 | 周期类型 | week/month/semester | 查询的时间范围 |
| 课程类型 | String | 否 | 课程代码 | 学校课程类型代码 | 筛选特定课程类型 |

### 5.2 输出数据格式
| 数据项 | 数据类型 | 格式 | 说明 |
|--------|----------|------|------|
| 综合评分 | Float | 数值，保留2位小数 | 0-100分的教学质量综合评分 |
| 教学活跃度 | Float | 数值，保留2位小数 | 权重25%，教学活跃度指标得分 |
| 学生参与度 | Float | 数值，保留2位小数 | 权重30%，学生参与度指标得分 |
| 教学效果 | Float | 数值，保留2位小数 | 权重35%，教学效果指标得分 |
| 教学创新 | Float | 数值，保留2位小数 | 权重10%，教学创新指标得分 |
| 院系排名 | Integer | 数值 | 在院系内的排名 |
| 校内排名 | Integer | 数值 | 在全校的排名 |
| 趋势对比 | Array | JSON数组 | 历史趋势对比数据 |

### 5.3 数据验证规则
- **格式验证：** 评分为0-100的数值，格式正确
- **业务验证：** 综合评分等于各维度加权得分之和
- **完整性验证：** 所有关键指标数据完整无缺失
