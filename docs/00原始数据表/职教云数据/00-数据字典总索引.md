# 职教云数据字典总索引

## 概述

本文档是职教云数据对接方案的完整数据字典索引，包含所有46个数据表的详细定义。数据表分为两大类别：

- **基础数据同步表**：用于从校方系统同步基础数据到职教云平台
- **教学过程数据推送表**：用于将职教云平台的教学过程数据推送给校方

## 数据流向说明

### 基础数据同步流向
```
校方系统 → 中间数据库 → 职教云平台
```

### 教学过程数据推送流向
```
职教云平台 → 中间数据库 → 校方数据分析平台
```

## 文档使用说明

1. **快速查找**：使用上表的搜索功能（Ctrl+F）按表名或中文描述搜索
2. **分类浏览**：按功能类别点击相应的文档链接查看详细信息
3. **字段说明**：每个数据字典文档都包含详细的字段说明和数据类型
4. **关联关系**：文档中会标注相关联的数据表，便于理解数据关系
