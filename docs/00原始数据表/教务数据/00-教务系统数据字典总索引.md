# 教务系统数据字典总索引

## 概述

本文档是教务系统数据字典的完整索引，包含所有25个数据表的详细定义。数据表分为六大功能模块，涵盖了教务系统的核心业务流程：

- **学生管理**：学生信息、异动、选课等相关数据
- **教学管理**：课程、教材、考试、实训等教学资源数据
- **课程安排**：课表、调停课、临时活动等教学安排数据
- **组织架构**：班级、教室、专业、院系等组织结构数据
- **基础数据**：校历、成绩等基础业务数据
- **数据集成**：外部数据源和集成数据

## 数据流向说明

### 学生数据流向
```
招生 → 学生基本信息 → 学生异动 → 选课 → 成绩管理 → 毕业
```

### 教学数据流向
```
教学计划 → 课程安排 → 教材管理 → 教学执行 → 考试管理 → 成绩录入
```

### 数据更新频率
- **基础数据**：学期更新（校历、专业、班级等）
- **学生数据**：实时更新（异动、选课等）
- **教学数据**：定期更新（课程安排、成绩等）
- **集成数据**：定时同步（外部数据源）

## 数据类型说明

### 常用数据类型
- **char**：固定长度字符串，用于代码、状态等字段
- **varchar**：可变长度字符串，用于名称、描述等字段
- **int**：整数类型，用于数量、序号等字段
- **decimal**：精确小数类型，用于成绩、分数等字段
- **datetime**：日期时间类型，用于时间戳字段

### 特殊字段类型
- **照片字段**：用于存储学生照片等图像数据
- **代码字段**：以 `_dm` 或 `_code` 结尾的字段
- **名称字段**：以 `_mc` 或 `_name` 结尾的字段
- **状态字段**：通常用 0/1 或特定字符表示不同状态
