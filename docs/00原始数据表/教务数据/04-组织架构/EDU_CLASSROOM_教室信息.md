# EDU_CLASSROOM 教室信息表

## 基本信息

- **表名**：`EDU_CLASSROOM`（根据课程安排表字段推导）
- **中文名称**：教室信息
- **用途**：存储教室基本信息，包括教室代码、名称、类型、容量、设备等详细信息
- **字段数量**：约15个（根据课程安排表字段推导）
- **数据类别**：教务系统 - 组织架构模块

| jsmc | 教室名称 | varchar | - | YES | NO | - | 教室名称 |

| rzrl | 容纳人数 | int | - | YES | NO | - | 标准容纳人数 |
| xqdm | 校区代码 | char | - | YES | NO | - | 所属校区 |
| xqmc | 校区名称 | varchar | - | YES | NO | - | 所属校区名称 |

## 字段详细说明

### 主键字段
- **jsdm**：教室代码，教室的唯一标识符

### 基本信息字段
- **jsmc**：教室名称，教室的完整名称
- **mj**：面积，教室的建筑面积

### 分类信息字段
- **jslx**：教室类型，教室的功能类型分类
- **rzrl**：容纳人数，教室的标准容纳人数

### 位置信息字段
- **xqdm**：校区代码，教室所属校区代码
- **xqmc**：校区名称，教室所属校区名称
- **lymc**：楼栋名称，教室所在楼栋
- **lc**：楼层，教室所在楼层

### 设备和状态字段
- **sbpz**：设备配置，教室配备的教学设备
- **zt**：状态，教室的使用状态
- **sfbm**：是否多媒体，是否配备多媒体设备
- **sfkf**：是否开放，是否对外开放使用

### 管理信息字段
- **glbm**：管理部门，教室的管理部门代码
- **glrx**：管理人，教室管理员姓名

### 其他字段
- **bz**：备注，其他需要说明的信息

## 教室代码说明

### 代码构成规则
- **校区标识**：代码前几位表示校区
- **楼栋标识**：包含楼栋信息
- **楼层标识**：包含楼层信息
- **房间编号**：具体的房间编号

### 代码示例
- **J101**：教学楼1楼01号教室
- **T201**：实验楼2楼01号教室
- **S301**：图书馆3楼01号教室

### 代码特点
- **唯一性**：全校范围内教室代码唯一
- **规范性**：遵循统一的编码规范
- **可读性**：代码包含有用信息
- **层次性**：体现层次结构关系

## 数据约束

1. **主键约束**：`jsdm` 字段必须唯一且非空
2. **检查约束**：`zt` 应在有效状态范围内
3. **检查约束**：`sfbm` 和 `sfkf` 只能是0或1
4. **数值约束**：`rzrl`、`lc`、`mj` 应为正数
5. **外键约束**：`xqdm` 应引用校区表的有效记录

## 使用示例

### SQL查询示例

```sql
-- 查询所有可用教室
SELECT jsmc, jslx, rzrl, xqmc, lymc, lc
FROM EDU_CLASSROOM
WHERE zt = 'KY' AND sfkf = '1'
ORDER BY xqmc, lymc, lc;

-- 查询多媒体教室
SELECT jsmc, rzrl, sbpz, glrx, jsmc
FROM EDU_CLASSROOM
WHERE sfbm = '1' AND zt = 'KY'
ORDER BY rzrl DESC;

-- 查询某校区的教室分布
SELECT jslx, COUNT(*) as room_count,
       SUM(rzrl) as total_capacity,
       AVG(rzrl) as avg_capacity
FROM EDU_CLASSROOM
WHERE xqdm = 'ZC' AND zt = 'KY'
GROUP BY jslx
ORDER BY room_count DESC;

-- 查询容量50人以上的教室
SELECT jsmc, jslx, rzrl, xqmc, lymc, lc, sbpz
FROM EDU_CLASSROOM
WHERE rzrl >= 50 AND zt = 'KY'
ORDER BY rzrl DESC;

-- 按楼层统计教室
SELECT lymc, lc, COUNT(*) as room_count,
       SUM(rzrl) as floor_capacity
FROM EDU_CLASSROOM
WHERE xqdm = 'ZC' AND zt = 'KY'
GROUP BY lymc, lc
ORDER BY lymc, lc;

-- 查询维修中的教室
SELECT jsmc, jslx, rzrl, glrx, bz
FROM EDU_CLASSROOM
WHERE zt = 'WX'
ORDER BY jsmc;
```

3. **设备匹配**：设备满足教学需求
4. **位置合理**：位置安排合理便利

1. **需求申请**：提交教室使用申请
2. **资源查询**：查询可用教室资源
