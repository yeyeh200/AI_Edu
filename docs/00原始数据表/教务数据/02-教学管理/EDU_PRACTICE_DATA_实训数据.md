# EDU_PRACTICE_DATA 实训数据表

## 基本信息

- **表名**：`EDU_PRACTICE_DATA`（原表名：`ODS_SXJXGCSJ`）
- **中文名称**：实训数据
- **用途**：存储学生实训教学数据，包括实训项目、课时安排、成绩评价等信息
- **字段数量**：11个
- **数据类别**：教务系统 - 教学管理模块

## 字段定义

| 字段名 | 原字段名 | 数据类型 | 长度 | 是否为空 | 主键 | 默认值 | 备注 |
|--------|----------|----------|------|----------|------|--------|------|
| zjsjwyxbs | 主键数据唯一性标识 | char | - | NO | YES | - | - |
| xxdm | 学校代码 | char | - | YES | NO | - | - |
| sxxmbh | 实训项目编号 | char | - | YES | NO | - | 即系统中的课程代码 |
| sszyh | 所属专业号 | char | - | YES | NO | - | 学生所在专业代码 |
| xh | 学号 | char | - | YES | NO | - | - |
| xsxm | 学生姓名 | char | - | YES | NO | - | - |
| xssxkss | 学生实训课时数 | numeric | - | YES | NO | - | 课程课表中排课课时数 |

## 字段详细说明

### 主键字段
- **zjsjwyxbs**：主键数据唯一性标识，确保每条实训记录的唯一性

### 学校和专业信息
- **xxdm**：学校代码，学校的统一标识代码
- **sszyh**：所属专业号，学生所在专业的代码

### 学生基本信息
- **xh**：学号，学生的唯一标识
- **xsxm**：学生姓名，实训学生的姓名

### 实训项目信息
- **sxxmbh**：实训项目编号，即系统中的课程代码，标识具体的实训项目

### 实训安排信息
- **xssxkss**：学生实训课时数，课程课表中排课的课时数
- **sxksrq**：实训开始日期，即第一节课排课的周节次
- **sxjsrq**：实训结束日期，即最后一节课排课的周节次

### 成绩和采集信息
- **sxjgzhpj**：实训结果综合评价，课程评价分数
- **sjcjsj**：数据采集时间，记录数据的采集时间

## 时间格式说明

### 实训日期格式
- **格式示例**：15周周一1-2节
- **组成要素**：
  - **周次**：15表示第15教学周
  - **星期**：周一表示星期一
  - **节次**：1-2表示第1-2节课

### 周次说明
- **起始周**：通常从第1周开始
- **结束周**：根据学期安排确定
- **实训周**：专门用于实践教学的时间段

## 课时数说明

### 课时计算标准
- **1课时**：通常为45分钟
- **2节连上**：1.5小时（90分钟）
- **半天实训**：4课时（3小时）
- **全天实训**：8课时（6小时）

### 实训课时构成
```
总实训课时数 = 排课课时数 = 实际实训课时数
```

### 课时安排类型
1. **集中实训**：连续多天集中进行
2. **分散实训**：每周固定时间分散进行
3. **分段实训**：分阶段进行实训
4. **弹性实训**：根据实际情况灵活安排

## 使用示例

### SQL查询示例

```sql
-- 查询某学生的实训记录
SELECT sxxmbh, xssxkss, sxksrq, sxjsrq, sxjgzhpj
FROM EDU_PRACTICE_DATA
WHERE xh = '2024001'
ORDER BY sxksrq;

-- 查询某实训项目的参与情况
SELECT xh, xsxm, sszyh, xssxkss, sxjgzhpj
FROM EDU_PRACTICE_DATA
WHERE sxxmbh = 'SX001'
ORDER BY sxjgzhpj DESC;

-- 统计某专业学生的实训情况
SELECT
    COUNT(*) as student_count,
    AVG(xssxkss) as avg_hours,
    AVG(sxjgzhpj) as avg_score,
    COUNT(CASE WHEN sxjgzhpj >= 60 THEN 1 END) as pass_count
FROM EDU_PRACTICE_DATA
WHERE sszyh = 'CS001';

-- 查询实训成绩分布
SELECT
    CASE
        WHEN sxjgzhpj >= 90 THEN '优秀'
        WHEN sxjgzhpj >= 80 THEN '良好'
        WHEN sxjgzhpj >= 70 THEN '中等'
        WHEN sxjgzhpj >= 60 THEN '及格'
        ELSE '不及格'
    END as grade_level,
    COUNT(*) as student_count
FROM EDU_PRACTICE_DATA
WHERE sxxmbh = 'SX001'
GROUP BY
    CASE
        WHEN sxjgzhpj >= 90 THEN '优秀'
        WHEN sxjgzhpj >= 80 THEN '良好'
        WHEN sxjgzhpj >= 70 THEN '中等'
        WHEN sxjgzhpj >= 60 THEN '及格'
        ELSE '不及格'
    END;

-- 查询实训时间安排冲突
SELECT sxxmbh, sxksrq, sxjsrq, COUNT(*) as conflict_count
FROM EDU_PRACTICE_DATA
WHERE sxksrq IS NOT NULL
GROUP BY sxxmbh, sxksrq, sxjsrq
HAVING COUNT(*) > 1;
```
