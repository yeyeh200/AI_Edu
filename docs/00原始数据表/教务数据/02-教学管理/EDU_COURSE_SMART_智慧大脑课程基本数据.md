## 字段详细说明

### 基本课程信息
- **CDDW**：承担单位，课程开设单位
- **KCHJDM**：课程/环节代码，课程或实践环节的唯一代码
- **KCHJMC**：课程/环节名称，课程或实践环节的完整名称
- **XF**：学分，课程学分值

### 课程分类信息
- **KCLB**：课程类别，课程的基本分类
- **KCSX**：课程属性，课程的具体属性
- **KCLX**：课程类型，课程的类型划分
- **KCBQ**：课程标签，用于课程标识和分类

### 智慧大脑属性编码字段
- **KCSXM**：课程属性码，智慧大脑平台的课程属性编码

### 课程特色标识字段

- **SFXQHZKF**：是否校企合作开放，校企合作开放课程标识

## 使用示例

### SQL查询示例

```sql
-- 查询所有线上课程
SELECT KCHJDM, KCHJMC, XF, CDDW, XSKCWZ
FROM EDU_COURSE_SMART
WHERE SFXSKC = '1'
ORDER BY KCHJMC;

-- 查询专业核心课程
SELECT KCHJMC, XF, KCLB, KCSX, CDDW
FROM EDU_COURSE_SMART
WHERE SFZYHXKSFXSKC = '1'
ORDER BY CDDW, KCHJMC;

-- 查询精品课程
SELECT KCHJMC, JPKC, KCLB, XF, CDDW
FROM EDU_COURSE_SMART
WHERE JPKC IS NOT NULL AND JPKC != ''
ORDER BY JPKC, KCHJMC;

-- 查询课程思政示范课程
SELECT KCHJMC, XF, KCLB, CDDW, SFKCSZSFK
FROM EDU_COURSE_SMART
WHERE SFKCSZSFK = '1'
ORDER BY CDDW, KCHJMC;

-- 查询校企合作开放课程
SELECT KCHJMC, KCLB, XF, CDDW, SFXQHZKF
FROM EDU_COURSE_SMART
WHERE SFXQHZKF = '1'
ORDER BY CDDW, KCHJMC;

-- 统计各类型课程数量
SELECT
    KCLB,
    COUNT(*) as total_count,
    SUM(CASE WHEN SFXSKC = '1' THEN 1 ELSE 0 END) as online_count,
    SUM(CASE WHEN SFZYHXKSFXSKC = '1' THEN 1 ELSE 0 END) as core_count,
    SUM(CASE WHEN SFKCSZSFK = '1' THEN 1 ELSE 0 END) as ideological_count
FROM EDU_COURSE_SMART
WHERE KCHJDM IS NOT NULL
GROUP BY KCLB
ORDER BY total_count DESC;
```
