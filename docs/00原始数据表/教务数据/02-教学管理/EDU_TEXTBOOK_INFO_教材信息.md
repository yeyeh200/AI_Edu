# EDU_TEXTBOOK_INFO 教材信息表

## 基本信息

- **表名**：`EDU_TEXTBOOK_INFO`（原表名：`V_BRW_LessonCord`）
- **中文名称**：教材信息
- **用途**：存储教材的基本信息，包括教材名称、ISBN、出版社、作者等详细信息
- **字段数量**：9个
- **数据类别**：教务系统 - 教学管理模块

## 字段定义

| 字段名 | 原字段名 | 数据类型 | 长度 | 是否为空 | 主键 | 默认值 | 备注 |
|--------|----------|----------|------|----------|------|--------|------|
| JC_kcdm | 课程ID | char | - | NO | YES | - | 教材代码 |
| KCmc | 教材名称 | char | - | YES | NO | - | - |
| ISBN | ISBN编号 | char | - | YES | NO | - | - |
| cbs | 出版社 | char | - | YES | NO | - | - |
| zz | 作者 | char | - | YES | NO | - | - |
| JCLB | 教材类别 | char | - | YES | NO | - | - |
| status | 状态 | CHAR | - | YES | NO | - | - |
| HJDJ | 获奖等级 | CHAR | - | YES | NO | - | - |
| BJ | 备注 | CHAR | - | YES | NO | - | - |

## 字段详细说明

### 主键字段
- **JC_kcdm**：课程ID，作为教材代码，主键，唯一标识教材

### 基本教材信息
- **KCmc**：教材名称，教材的完整名称
- **ISBN**：ISBN编号，国际标准书号
- **cbs**：出版社，教材出版单位名称
- **zz**：作者，教材作者或编者

### 教材分类字段
- **JCLB**：教材类别，区分教材类型
- **HJDJ**：获奖等级，教材获得的奖项等级

### 状态和备注字段
- **status**：状态，教材的使用状态
- **BJ**：备注，其他说明信息

## 教材类别说明

### 常见教材类别（JCLB）
- **统编教材**：国家统一编写的教材
- **规划教材**：列入规划教材系列的教材
- **自编教材**：学校自行编写的教材
- **引进教材**：从国外引进的教材
- **实验教材**：实验课程专用教材
- **实训教材**：实训课程专用教材
- **参考教材**：辅助参考资料

### 教材分类标准
1. **编写来源**：统编、自编、引进等
2. **使用范围**：公共、专业、选修等
3. **课程类型**：理论、实践、实验等
4. **出版级别**：国家级、省部级、校级等

## 状态说明

### 教材状态（status）
| 状态值 | 说明 | 含义 |
|--------|------|------|
| 1 | 启用 | 教材正在使用中 |
| 0 | 停用 | 教材已停用 |
| 2 | 待审 | 教材正在审核中 |
| 3 | 备案 | 教材已备案待用 |

### 状态转换流程
1. **新增**：待审 → 启用
2. **更新**：启用 → 待审 → 启用
3. **停用**：启用 → 停用
4. **恢复**：停用 → 待审 → 启用

## 获奖等级说明

### 常见获奖等级（HJDJ）
- **国家级**：国家级优秀教材
- **省部级**：省部级别优秀教材
- **校级**：校级优秀教材
- **行业级**：行业协会评选的优秀教材
- **其他**：其他奖项或荣誉

### 奖项类型
1. **教学成果奖**：教学成果方面的奖励
2. **优秀教材奖**：专门针对教材的奖项
3. **精品教材**：精品课程配套教材
4. **规划教材**：各类规划教材项目

## ISBN规范说明

### ISBN格式
- **ISBN-10**：10位数字的旧格式
- **ISBN-13**：13位数字的新格式（现行标准）
- **格式示例**：978-7-115-12345-6

### ISBN组成
1. **前缀码**：978或979
2. **组号**：国家、语言或地区代码
3. **出版者码**：出版社代码
4. **书名码**：特定书名的代码
5. **校验码**：最后一位校验数字

- **专业出版社**：专业领域出版社

- **科技出版社**：科学技术类出版社
- **文艺出版社**：文学艺术类出版社

### 知名出版社
1. **高等教育出版社**：高等教育教材主要出版商
2. **人民教育出版社**：基础教育教材权威出版社
3. **机械工业出版社**：工程技术类专业出版社
4. **清华大学出版社**：知名大学出版社

## 数据约束

1. **主键约束**：`JC_kcdm` 字段必须唯一且非空
2. **唯一约束**：`ISBN` 字段应保持唯一性
3. **检查约束**：`status` 应符合预设状态值
4. **格式约束**：`ISBN` 应符合标准ISBN格式

## 使用示例

### SQL查询示例

```sql
-- 查询所有启用状态的教材
SELECT JC_kcdm, KCmc, cbs, zz, ISBN, JCLB, HJDJ
FROM EDU_TEXTBOOK_INFO
WHERE status = '1'
ORDER BY KCmc;

-- 查询某出版社的教材
SELECT JC_kcdm, KCmc, zz, ISBN, HJDJ
FROM EDU_TEXTBOOK_INFO
WHERE cbs = '高等教育出版社' AND status = '1'
ORDER BY KCmc;

-- 查询获奖教材
SELECT KCmc, cbs, zz, HJDJ, JCLB
FROM EDU_TEXTBOOK_INFO
WHERE HJDJ IS NOT NULL AND status = '1'
ORDER BY HJDJ, KCmc;

-- 按教材类别统计
SELECT
    JCLB,
    COUNT(*) as book_count,
    COUNT(CASE WHEN HJDJ IS NOT NULL THEN 1 END) as award_count
FROM EDU_TEXTBOOK_INFO
WHERE status = '1'
GROUP BY JCLB
ORDER BY book_count DESC;

-- 查询特定作者的教材
SELECT KCmc, ISBN, cbs, JCLB, HJDJ
FROM EDU_TEXTBOOK_INFO
WHERE zz LIKE '%张三%' AND status = '1'
ORDER BY KCmc;
```

### 数据插入示例

```sql
INSERT INTO EDU_TEXTBOOK_INFO (
    JC_kcdm, KCmc, ISBN, cbs, zz, JCLB, status, HJDJ
) VALUES (
    'TB001', '数据结构（C语言版）', '978-7-115-12345-6',
    '人民邮电出版社', '严蔚敏', '统编教材', '1', '国家级'
);
```

### 数据更新示例

```sql
-- 更新教材状态
UPDATE EDU_TEXTBOOK_INFO
SET status = '0'
WHERE JC_kcdm = 'TB001';

-- 更新教材获奖信息
UPDATE EDU_TEXTBOOK_INFO
SET HJDJ = '省部级'
WHERE JC_kcdm = 'TB002';
```

## 业务逻辑说明

### 教材选用原则
1. **适用性**：适合课程教学目标
2. **先进性**：内容先进，反映学科发展
3. **稳定性**：版本相对稳定，便于教学
4. **经济性**：价格合理，学生可承受

### 教材评审流程
1. **教师推荐**：任课教师推荐适用教材
2. **专家评审**：组织专家进行评审
3. **学院审核**：学院教学委员会审核
4. **学校批准**：学校教务部门批准
5. **备案使用**：正式备案投入使用

### 质量控制
1. **内容审查**：确保教材内容科学准确
2. **版权检查**：确保教材版权合法
3. **更新及时**：及时更新过时内容
4. **评价反馈**：收集使用反馈意见

## 数据维护建议

### 定期维护
1. **信息更新**：及时更新教材信息
2. **状态管理**：根据使用情况更新状态
3. **版本管理**：管理教材版本更新
4. **库存同步**：与库存信息同步

### 异常处理
1. **重复教材**：检查并处理重复的教材记录
2. **ISBN错误**：纠正错误的ISBN号码
3. **状态异常**：处理状态设置异常的情况
4. **信息不全**：补充缺失的教材信息
